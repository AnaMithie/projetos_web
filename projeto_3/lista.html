listas não ordenadas

a tag ul (unordered list - lista não ordenada) representa uma lista de itens sem ordem rigida, por exemplo ingrdientes em uma receita.

A tag li (list itens- itens da lista) representada por um iten que faz parte de uma lista.

Exemplo de lista

  • Primeiro item
  • Segundo item
    • subitem

Lista ordenadas

A tag ol( ordered list - lista ordenada) representada uma lista de itens numerados sequencialmente.

Exemplo de lista ordenada

  1. primwiro item
  2. segundo item

atributos

reversed (ordem reversa)

Exibe os itens da lista ordenada reversa.

  1. item mais importante
  2. item menos importante

start (inicio)
especifica o valor inicial para numeração da lista.

  1. item 5
  2. item 4

type (tipo)
indica o tipo de numeração:
a- indica letras minusculas:
A- indica letras maiusculas:
i- indica algarismos romanos minusculos I- indica algarismos romanos maiusculos

  1. primeiro item
  2. segundo item
